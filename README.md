# RealEstate – Find Your Dream Home

A professional, responsive real estate website designed to help users find their ideal residential and commercial properties. This project features a modern UI, property filtering, and a functional contact system.

## ✨ Features
- **Responsive Design:** Optimized for all devices (desktop, tablet, and mobile).
- **Interactive UI:** Smooth transitions and hover effects for a premium feel.
- **Property Listings:** Browse featured and all properties with detailed cards.
- **Dynamic Filtering:** Filter properties by type (Apartment, Villa, House, etc.).
- **Property Details:** Dedicated pages for in-depth property information.
- **Form Validation:** Functional contact and enquiry forms with client-side validation.
- **Modern Styling:** Built using CSS variables for easy customization and consistent branding.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styles using modern layouts (Flexbox/Grid) and CSS variables.
- **JavaScript (ES6+):** Dynamic filtering logic and form validation.

## 📁 Project Structure
```text
real-estate-website/
├── index.html            # Home page
├── about.html            # Company information
├── properties.html       # All property listings
├── property-detail.html  # Specific property details
├── contact.html          # Contact and enquiry page
├── styles.css            # Main stylesheet
└── script.js             # Form and filter logic
```

## ⚙️ Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/real-estate-website.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd real-estate-website
   ```
3. **Open the project:**
   Simply open `index.html` in your favorite web browser to view the site.
